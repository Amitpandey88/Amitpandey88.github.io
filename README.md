# theamit.me — Amit Pandey's Portfolio

Personal portfolio website for **Amit Pandey**, a Computer & Data Science student exploring AI, Data Engineering, and Software Development.

🌐 **Live site:** [theamit.me](https://theamit.me)

## Tech Stack

- **HTML5** — semantic, accessible markup
- **CSS3** — dark glassmorphism design with gradient accents
- **Vanilla JavaScript** — typed animation, scroll effects, mobile nav, reveal-on-scroll

## Features

- Responsive, mobile-friendly layout
- Typing animation hero section
- Skills showcase (AI/ML, Data, Software Dev, Cloud & Tools)
- Project showcase section (coming soon)
- Journey / roadmap timeline
- Contact links (GitHub, LinkedIn, Email)

## Sections

| # | Section | Description |
|---|---------|-------------|
| 01 | About | Background and introduction |
| 02 | Skills | Tech stack grouped by category |
| 03 | Projects | Polished projects (in progress) |
| 04 | Journey | Roadmap timeline |
| 05 | Contact | GitHub, LinkedIn, and Email links |

## Local Development

No build step required. Open `index.html` directly in a browser, or serve with any static file server:

```bash
npx serve .
```
